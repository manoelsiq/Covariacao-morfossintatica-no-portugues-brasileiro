Nesta pasta, é possível encontrar as planilhas com os dados utilizados nesta pesquisa. As planilhas são de três tipos:

1. Tipo 1: planilhas com todas as ocorrências das variáveis, contendo os excertos dos quais as ocorrências foram retiradas. São as planilhas det (uso variável de artigo antes de possessivos pré-nominais), pro (pronomes pessoais de 2PS em posição de sujeito), cli (clíticos de 2PS) e pos (possessivos de 2PS). São as planilhas utilizadas para análises categóricas e para a extração de taxas individuais.

2. Tipo 2: planilha com as taxas de uso das variantes de aplicação, com log odds, dados extralinguísticos e outras informações. É a planilha utilizada para a análise de covariação. Seu nome é "dados". 

3. Tipo 3: planilha com os dados sociais de cada um dos informantes. São considerados apenas os dados sociais mobilizados para a tese.
