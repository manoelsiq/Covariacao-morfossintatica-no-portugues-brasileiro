Nesta pasta, é possível encontrar os scripts utilizados para as análises de covariação: teste de correlação e técnica de cluster. Há dois scripts de cada:

1. Script de Correlação: teste de correlação com as frequências individuais de uso;
2. Script de Correlação (informante aleatório): teste de correlação com os log-odds individuais;
3. Script de Cluster: análise de cluster por k-medoids a partir apenas das frequências individuais de uso;
4. Script de Cluster (informações sociais): análise de cluster por k-medoids considerando também as informações sociais dos falantes.

Para todos esses scripts, deve-se utilizar planilha "dados".
