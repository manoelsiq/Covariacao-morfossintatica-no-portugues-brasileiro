Nesta pasta, são encontrados três scripts para análise estatística descritiva e inferencial para variáveis categóricas.

1. Script de análise categórica: apresenta as distribuições das variáveis dependentes considerando: distribuição geral, por amostra, por deslocamento e por tempo no curso;

2. Script de frequência individual: apresenta as distribuições das variáveis dependentes por meio dos usos feitos por cada falante;

3. Script de variáveis sociais: apresenta as distribuições das variáveis dependentes considerando gênero e idade.
